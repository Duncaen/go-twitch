# go-twitch

[![Build Status](https://travis-ci.org/mrshankly/go-twitch.svg?branch=master)](https://travis-ci.org/mrshankly/go-twitch)

go-twitch is a Go client library for accessing the [Twitch V5 API](https://dev.twitch.tv/docs).
